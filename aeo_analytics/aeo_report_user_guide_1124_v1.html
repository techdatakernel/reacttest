1종합 점수 이해하기

🤔 종합 점수가 뭔가요?

여러분의 웹페이지가 ChatGPT, Perplexity, Google AI 같은 AI 검색엔진에서 얼마나 잘 노출될 수 있는지를 0~100점 사이로 보여주는 점수입니다.

쉽게 말해, AI가 여러분의 페이지를 추천할 확률이 얼마나 되는지를 나타냅니다.

90~100점

우수 - AI가 적극 추천

75~89점

양호 - 추천 가능성 높음

60~74점

보통 - 개선 필요

0~59점

미흡 - 즉시 개선 필요

💡 실제 예시

상황: "운전자보험 어디가 좋은지 추천해줘"라고 ChatGPT에 질문했을 때
69.4점 페이지: 상위 5개 추천 중 3번째 정도로 나올 가능성
85점 페이지: 상위 5개 추천 중 1~2번째로 나올 가능성

🔍 BM25 키워드 점수 이해하기

2BM25가 뭔가요?

🤔 어려운 용어는 잊으세요!

BM25는 "핵심 키워드가 얼마나 잘 들어가 있나?"를 측정하는 점수입니다.

예를 들어, "운전자보험"이라는 키워드로 검색했을 때, 여러분의 페이지에 이 단어가 적절하게 반복되어 있는지 확인합니다.

💡 음식점으로 비유하면?

메뉴판 점수라고 생각하면 됩니다.

- 김치찌개 전문점 메뉴판에 "김치찌개"가 5번 나옴 → 높은 점수 ✅
- 김치찌개 전문점인데 "맛있는 음식"만 쓰여있음 → 낮은 점수 ❌

❌ 낮은 점수 예시 (20점)

"저희는 최고의 보험을 제공합니다. 고객님께 가장 좋은 상품을 추천드립니다."

✅ 높은 점수 예시 (80점)

"운전자보험 추천을 찾으신다면 DB손해보험 운전자보험이 좋습니다. 운전자보험 가입 시 자동차사고와 보행사고 모두 보장받을 수 있습니다."

💡 개선 팁

보고서의 "BM25 키워드 분석" 테이블을 보세요:

  • 빈도(TF): 해당 키워드가 몇 번 나왔는지
  • 위치: 제목, 본문 어디에 있는지
  • 최소성: 문서 전체에서 얼마나 집중적으로 다뤄지는지

개선 방법: 점수가 낮은 키워드를 제목과 본문 초반부에 2~3번 더 추가하세요.

🧠 시맨틱 유사도 이해하기

3시맨틱 유사도가 뭔가요?

🤔 또 어려운 용어네요?

걱정마세요! "의미적으로 얼마나 관련있나?"를 측정하는 점수입니다.

단순히 단어가 있는지가 아니라, 내용이 질문과 얼마나 잘 맞는지를 봅니다.

💡 실생활 비유

질문: "저렴한 가격에 맛있는 김치찌개집 추천해줘"

낮은 점수 (40점):
"우리 집은 김치찌개가 있습니다. 위치는 강남역입니다."
→ 단어만 있고 실제 질문에 대한 답이 없음

높은 점수 (85점):
"1인분 7,000원의 합리적인 가격에 국내산 김치를 사용한 진한 국물 맛이 일품입니다. 김치찌개 단골 손님이 많은 이유죠."
→ 가격, 맛, 이유까지 구체적으로 답변

항목 의미 점수 기준
주제 일치도 질문 주제와 얼마나 맞는지 90점 이상: 완벽히 일치
의미적 관련성 연관된 개념을 다루는지 85점 이상: 관련 개념 포함
맥락 이해도 질문 의도를 파악했는지 80점 이상: 의도 정확히 파악
정보 충족도 필요한 정보를 제공하는지 90점 이상: 정보 완벽 제공
💡 개선 팁

시맨틱 점수를 높이려면:

  • 질문의 숨은 의도를 파악하세요 (예: "추천"은 장단점 비교가 필요)
  • 구체적인 숫자, 사례, 근거를 추가하세요
  • 관련된 유사 질문들도 함께 답변하세요

❓ FAQ 구조 점수 이해하기

4FAQ가 왜 중요한가요?

🤔 FAQ = Frequently Asked Questions (자주 묻는 질문)

AI는 질문-답변 형식으로 정리된 내용을 가장 선호합니다!

왜냐하면 사용자들도 AI에게 질문 형태로 물어보기 때문이죠.

✅ AI가 좋아하는 FAQ 형식

Q: 운전자보험은 꼭 가입해야 하나요?
A: 네, 자동차보험으로 커버되지 않는 벌금이나 형사합의금을 보장받을 수 있습니다.

Q: 운전자보험 가입 시 고려할 점은?
A: 보장 범위(벌금, 형사합의금)와 자기부담금을 꼭 확인하세요.

❌ AI가 이해하기 어려운 형식

운전자보험 안내
운전자보험은 좋은 상품입니다. 여러 혜택이 있으며 가입을 권장합니다. 자세한 내용은 상담을 통해 확인하세요.

💡 FAQ 개선 실전 팁

보고서에서 "AI 제안 질문 리스트"를 확인하세요:

  • AI가 자동으로 고객들이 궁금해할 7~10개 질문을 생성했습니다
  • 각 질문에 대한 답변을 2~3문장으로 작성하세요
  • 제목에 "자주 묻는 질문" 또는 "FAQ"를 명시하세요
❌ 개선 전 (FAQ 없음)

긴 설명문으로만 구성
→ AI가 핵심 내용 파악 어려움
→ FAQ 점수: 0/20점

✅ 개선 후 (FAQ 추가)

5개의 Q&A로 핵심 정리
→ AI가 즉시 답변 가능
→ FAQ 점수: 18/20점

📌 관련성 증거 이해하기

5관련성 증거가 뭔가요?

🤔 AI에게 "근거"를 보여주는 것

"우리 페이지가 정말 이 주제에 대해 전문성이 있다"는 것을 객관적 데이터로 증명하는 섹션입니다.

💡 실제 적용 예시

키워드 관련성: "운전자보험" 단어가 본문에 10번, 메타 설명에 1번 포함
의미적 관련성: 질문과 85% 일치하는 내용 구성

💡 이 점수를 높이려면?
  • 제목에 핵심 키워드를 반드시 넣으세요
  • 첫 문단에 핵심 내용을 요약하세요
  • 구체적 수치와 사례를 포함하세요 (예: "2024년 기준", "7,000원")

6우선순위별 개선 방법

🎯 가장 효과적인 개선 순서

보고서의 "개선 권고사항" 섹션은 우선순위별로 정렬되어 있습니다.

✅ 즉시 실행 가능한 개선 로드맵

  1. 1단계 (오늘 당장!): FAQ 섹션 추가
    • AI 제안 질문 리스트에서 5~7개 선택
    • 각 질문에 2~3문장으로 답변 작성
    • 페이지 하단에 "자주 묻는 질문" 섹션으로 추가
  2. 2단계 (이번 주 내): 키워드 밀도 조정
    • BM25 분석표에서 낮은 점수 키워드 확인
    • 해당 키워드를 제목과 첫 문단에 추가
    • 자연스럽게 2~3번 더 반복
  3. 3단계 (이번 달 내): 구조화된 데이터 마크업
    • FAQ Schema 마크업 추가 (개발팀 협조)
    • 메타 태그 최적화
💡 개선 전후 비교 예시

  • FAQ 추가 → +8점
  • 키워드 최적화 → +5점
  • 의미적 관련성 강화 → +3점

개선 후 예상 점수: 85.4점 (양호) ✨

💬 이용자들이 자주 묻는 질문

Q1: 점수가 낮게 나왔는데 실제 트래픽은 많아요. 왜 그런가요?

A: 기존 SEO(네이버, 구글 검색)와 AEO(AI 검색)는 다른 기준입니다. 기존 검색에서는 인기가 많아도, AI 검색 최적화가 안 되어 있으면 AI가 추천하지 않을 수 있습니다. 향후 AI 검색 점유율이 증가하므로 지금부터 대비하세요.

Q2: 개선 후 얼마나 지나야 효과가 나타나나요?

A: - FAQ 추가: 즉시 효과 (1~3일 내)
- 키워드 최적화: 1~2주 후
- 구조적 변경: 2~4주 후
개선 후 1주일 뒤 재분석해서 점수 변화를 확인하세요.

Q3: 모든 페이지를 분석해야 하나요?

A: 우선순위를 정하세요:
1순위: 메인 랜딩 페이지, 전환율 높은 페이지
2순위: 트래픽 많은 상위 10개 페이지
3순위: 신규 발행 페이지

Q4: IT 개발 없이도 개선할 수 있나요?

A: 네! 80% 이상은 콘텐츠만 수정해도 됩니다:
- FAQ 추가 (복사+붙여넣기)
- 키워드 추가 (텍스트 편집)
- 내용 보강 (에디터에서 수정)
개발이 필요한 것은 Schema 마크업 정도입니다.
