1. 종합 점수 이해하기

종합 점수란?

종합 점수는 여러분의 웹페이지가 AI 검색에 얼마나 최적화되어 있는지를 100점 만점으로 보여줍니다. 이 점수는 세 가지 항목을 합산한 결과입니다.

69.4점
분석 항목 점수 만점 평가
BM25 키워드 (단어 찾기) 33점 40점 우수 ✓
시맨틱 유사도 (의미 이해) 37점 43점 우수 ✓
FAQ 구조 (질문-답변) 5점 20점 개선 필요 ✗

실생활 비유

종합 점수를 시험 점수라고 생각해보세요:

  • 90점 이상: A+ (탁월함) - AI가 적극 추천합니다
  • 70~89점: B+ (우수함) - 좋은 상태이나 개선 여지가 있습니다
  • 50~69점: C+ (보통) - 개선이 필요합니다
  • 50점 미만: D (부족) - 많은 개선이 필요합니다

점수 향상 팁

위 예시에서 FAQ 구조 점수가 낮습니다. 질문-답변 형식의 콘텐츠를 추가하면 점수가 크게 향상됩니다!

2. BM25 키워드 분석 (단어 찾기)

BM25란 무엇인가요?

BM25는 "단어 찾기" 방식입니다. 사용자가 검색한 단어가 여러분의 페이지에 얼마나 많이, 어떤 위치에 있는지를 확인합니다.

쉽게 말해, 단어 매칭 점수입니다.

실제 예시로 이해하기

사용자 검색: "운전자보험 추천 좋은 곳"

페이지 분석 결과:

  • 운전자보험 - 페이지 제목에 있음 (높은 점수) ✓
  • 보험료 - 본문에 여러 번 나옴 (중간 점수)
  • 자동차 사고 - 본문에 몇 번 나옴 (중간 점수)
  • 간병인 - 본문에 있지만 덜 중요함 (낮은 점수)
  • 보장 - 본문에 적게 나옴 (낮은 점수)

🎯 BM25 점수 향상 방법

1. 제목에 중요한 키워드 넣기: 제목에 있는 단어는 가장 높은 점수를 받습니다.
2. 반복하되 자연스럽게: 중요한 단어를 본문에 여러 번 넣되, 너무 많으면 오히려 점수가 낮아집니다.
3. 첫 부분에 배치: 페이지 앞쪽에 나오는 단어가 더 중요하게 평가됩니다.

위치 중요도 예시
제목 (H1, H2) 매우 높음 ★★★ "운전자보험 비교 가이드"
첫 문단 높음 ★★ "운전자보험은 자동차 사고 시..."
본문 중간 ★ "보험료 비교를 통해..."
3. 시맨틱 유사도 (의미 이해하기)

시맨틱 유사도란?

시맨틱(Semantic)은 "의미"라는 뜻입니다. 시맨틱 유사도는 단순히 단어를 찾는 것이 아니라, 페이지의 내용이 사용자의 질문과 의미적으로 얼마나 관련있는지를 AI가 판단한 결과입니다.

ChatGPT나 Gemini 같은 AI는 단어 그대로가 아니라 "무엇을 말하고 있는지"를 이해합니다.

실제 예시로 이해하기

사용자 질문: "운전자보험 추천 좋은 곳"

두 페이지 비교:

페이지 A (낮은 시맨틱 점수):

  • "운전자보험" 단어만 10번 반복
  • 실제 추천이나 비교 내용 없음
  • → BM25는 높지만 시맨틱 점수는 낮음

페이지 B (높은 시맨틱 점수):

  • 다양한 보험 상품 비교
  • 각 보험의 장단점 설명
  • 가격대별 추천
  • 실제 선택 방법 안내
  • → 사용자 질문에 완벽하게 답함

📊 시맨틱 점수의 4가지 구성 요소

1. 주제 일치도 (10점): 질문과 페이지 주제가 얼마나 일치하나?
2. 의미적 연관성 (9점): 질문과 관련된 내용을 다루고 있나?
3. 핵심 어휘도 (8점): 중요한 전문 용어를 적절히 사용했나?
4. 정보 충족도 (10점): 사용자가 원하는 정보를 충분히 제공하나?

시맨틱 점수 향상 팁

  • 질문에 직접 답하기: "어떤 운전자보험이 좋나요?" → "추천하는 3가지 보험은..."
  • 관련 정보 포함: 가격, 보장 범위, 선택 기준 등
  • 구체적인 예시: "A사는 월 3만원, B사는 월 2.5만원"
  • 자연스러운 문장: 키워드만 나열하지 말고 완전한 문장으로 작성
4. FAQ 구조 분석 (질문-답변 형식)

FAQ 구조란?

FAQ(Frequently Asked Questions)는 "자주 묻는 질문"입니다. AI 검색 엔진은 질문-답변 형식으로 되어 있는 콘텐츠를 특히 좋아합니다.

왜냐하면 사용자가 질문하면 AI가 바로 답변을 찾아서 제시할 수 있기 때문입니다.

나쁜 예 (FAQ 구조 없음)

운전자보험은 자동차 사고 시 필요한 보험입니다. 보험료는 3만원 정도이고 보장 범위는 다양합니다. 여러 상품이 있으며 선택 시 비교가 필요합니다.

✗ 정보는 있지만 질문-답변 형식이 아니라서 AI가 찾기 어렵습니다.

좋은 예 (FAQ 구조 있음)

Q. 운전자보험이란 무엇인가요?

A. 운전자보험은 자동차 사고 시 운전자에게 발생하는 비용을 보장하는 보험입니다.

Q. 보험료는 얼마나 하나요?

A. 평균 월 3만원 정도이며, 보장 범위에 따라 2만원~5만원 사이입니다.

Q. 어떤 보험이 좋나요?

A. 본인의 운전 패턴과 예산에 맞춰 선택하시면 됩니다. 추천하는 3가지 상품은...

✓ 질문-답변 형식으로 되어 있어 AI가 쉽게 찾고 인용할 수 있습니다!

💎 FAQ 구조의 장점

1. AI가 찾기 쉬움: 질문에 바로 매칭되는 답변을 제공
2. 높은 인용률: ChatGPT 등이 여러분의 답변을 직접 인용
3. 사용자 만족도: 명확한 구조로 사용자도 이해하기 쉬움
4. 검색 최적화: 구글 등 기존 검색엔진에도 유리

FAQ 작성 팁

  • 실제 질문 형태로: "운전자보험이란?" 보다 "운전자보험이란 무엇인가요?"가 좋음
  • 7~10개 작성: 너무 적으면 효과가 없고, 너무 많으면 복잡함
  • 구체적인 답변: "다양합니다" 보다 "3가지 주요 유형은..."
  • 사용자 입장에서: 실제로 자주 묻는 질문 위주로 구성
5. 쿼리 확장 결과 (관련 질문들)

쿼리 확장이란?

쿼리 확장(Query Expansion)은 사용자가 입력한 질문과 관련된 다른 질문들을 찾아내는 것입니다.

예를 들어, "운전자보험 추천"을 검색한 사람은 다음과 같은 질문도 할 가능성이 높습니다:

쿼리 확장 예시

원래 질문: "운전자보험 추천 좋은 곳"

관련 질문들:

  • 운전자보험 추천 비교 사이트는 어디인가요?
  • 운전자보험 가입 시 고려해야 할 사항은 무엇인가요?
  • 운전자보험과 자동차보험의 차이점은 무엇인가요?
  • DB손해보험 운전자보험의 특징은 무엇인가요?
  • 운전자보험 할인 혜택은 어떤 것이 있나요?

🎯 쿼리 확장이 중요한 이유

사용자들은 같은 내용을 다양한 방식으로 질문합니다. 여러분의 페이지가 이런 다양한 질문에 모두 답할 수 있다면, AI가 여러분의 페이지를 더 자주 추천하게 됩니다.

활용 방법

보고서의 쿼리 확장 결과를 보고, 여러분의 페이지에 없는 질문들을 찾아보세요. 그 질문들에 대한 답변을 추가하면 더 많은 검색에 노출될 수 있습니다!

6. 관련성 증거 (얼마나 관련있나)

관련성 증거란?

관련성 증거는 여러분의 페이지가 사용자의 질문과 얼마나 관련있는지를 증명하는 부분입니다. 두 가지 방식으로 관련성을 평가합니다:

평가 방식 설명 예시
키워드 관련성 검색 단어가 페이지에 있는지 "운전자보험" 단어가 10번 나옴 → 10/10
의미 관련성 내용이 질문과 관련있는지 보험 비교, 추천, 가격 정보 포함 → 10/10

비율로 보는 관련성

예시 결과:

  • 키워드 관련성: 10/10 (매우 좋음)
  • 의미 관련성: 8/10 (좋음)

해석: 단어는 충분히 포함되어 있지만, 의미적으로 조금 더 깊이 있는 내용이 필요합니다.

관련성 향상 방법

  • 키워드 관련성이 낮을 때: 중요한 단어를 제목과 본문에 추가
  • 의미 관련성이 낮을 때: 질문에 대한 구체적인 답변과 예시 추가
  • 둘 다 낮을 때: 페이지 주제를 다시 검토하고 재작성 고려
7. AEO 개선 권고사항 (무엇을 고쳐야 할까)

개선 권고사항이란?

분석 결과를 바탕으로 AI가 자동으로 제안하는 개선 방법입니다. 이 항목들을 따라 페이지를 수정하면 점수가 향상됩니다.

📋 개선 권고사항의 3가지 유형

1. 누락된 정보: 페이지에 없지만 추가하면 좋을 내용
2. 비교 분석 정보: 여러 옵션을 비교하는 내용
3. 실행 핵심: 사용자가 바로 행동할 수 있는 구체적인 가이드

실제 권고사항 예시

누락된 정보

필요 이유: 사용자들이 선택 시 참고할 수 있는 실제 정보
예상 효과: 고객 신뢰도 증가 및 선택 유도

비교 분석 정보

필요 이유: 다른 보험사와의 비교를 통해 선택 기준 제공
예상 효과: 소비자가 더 쉽게 선택할 수 있음

실행 핵심

필요 이유: 고객이 실제로 신청하는 방법을 단계별로 안내
예상 효과: 가입 전환율 상승

예상 점수 증가

권고사항을 모두 반영하면 예상되는 점수 증가:

  • BM25: +5점 (더 많은 관련 키워드 포함)
  • 시맨틱: +8점 (의미적으로 더 풍부한 내용)
  • FAQ: +3점 (질문-답변 추가)

→ 총 +16점 향상 가능! (69.4점 → 85.4점)

8. AI 질문 리스트 (자주 묻는 질문 예측)

AI 질문 리스트란?

AI가 여러분의 페이지를 분석해서 사용자들이 자주 검색할 만한 질문 7~10개를 자동으로 생성해줍니다. 이 질문들은 실제 검색 데이터와 콘텐츠 분석을 바탕으로 만들어집니다.

AI가 생성한 질문 예시

  1. 운전자보험 추천 비교 사이트는 어디인가요?
  2. 운전자보험 가입 시 고려해야 할 사항은 무엇인가요?
  3. 운전자보험과 자동차보험의 차이점은 무엇인가요?
  4. 운전자보험 할인 혜택은 어떤 것이 있나요?
  5. 간병인 로봇 활용이 가능한가요?
  6. 운전자보험과 자동차보험의 차이점은 무엇인가요?
  7. 운전자보험 기간 시 고려해야 할 보장은?
  8. DB손해보험 운전자보험의 특징은 무엇인가요?
  9. 가족 단지 없는 운전자보험은 어떤 것이 있나요?
  10. 운전자보험 할인이 어떤 것이 있나요?

💡 AI 질문 리스트의 활용 방법

  1. 현재 페이지 확인: 생성된 질문들에 대한 답변이 이미 있는지 확인
  2. 누락된 내용 파악: 답변이 없는 질문을 찾아냄
  3. 콘텐츠 추가: 누락된 질문에 대한 답변을 FAQ 형식으로 추가
  4. 재분석: 내용을 추가한 후 다시 분석해서 점수 향상 확인

효과적인 활용 팁

  • 우선순위 설정: 질문 리스트 중 가장 많이 검색될 것 같은 3~5개 선택
  • 자연스러운 통합: 기존 내용에 자연스럽게 추가
  • 구체적인 답변: 각 질문에 최소 2~3문장 이상의 답변 작성
  • 정기 업데이트: 한 달에 한 번 새로운 질문 생성 및 추가

단계별 개선 프로세스

  1. 현재 상태 파악: 종합 점수와 각 항목별 점수를 확인하여 강점과 약점 파악
  2. 우선순위 선정: 가장 낮은 점수의 항목부터 개선 (보통 FAQ 구조가 가장 낮음)
  3. AI 질문 리스트 활용: 생성된 질문 중 없는 내용을 FAQ로 추가
  4. 권고사항 반영: 누락된 정보와 실행 핵심 내용 추가
  5. 재분석 및 확인: 수정 후 다시 분석하여 점수 향상 확인
